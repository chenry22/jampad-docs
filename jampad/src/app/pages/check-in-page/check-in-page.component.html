Check Ins

When an event is active, you will see a special option to "Check In" instead of the normal RSVP button. This is a way to let other JamPad users see how active an event is.

To check in, you must be within a reasonable distance from the event. If you are too far away to check in, you will get a notice telling you this.

Whenever you check into a verified event you will earn 30 FrogCoin!
