Map View

Find it on JamPad.

The map view is the core functionality of JamPad. At the bottom of the map is a search button. Any areas visible on your screen will be searched for events based on your current filters. Each event will show up as a map marker. Clicking on any of these markers will pull up a scrollable view with previews of events that appeared from your search!


Map Filters

If you want to change the scope of your search, you can pull up your search filters. Here you can change the date, cost, and age range you'd like to search for. You can also specify any event tags you'd like your search to consider.


Location Search

At the top of the screen is a search bar. You can use this to search for any events happening at specific locations. Your searched location will appear as a custom marker- click this to see all registered events from that venue!

On this page you can also create a new event at your searched location. This will pull up the normal event creator page but with the address filled in.
