Event Creator

Share what's happening!

The event creator allows you to share new events with the JamPad community! To post an event, you only have to provide a valid address and date (though there are many other fields which you can fill out to provide more info about the event).

If applicable, you can set your event to be private. Private events will not show up in public feeds and the event location will only be shown to users who are RSVPed. All RSVPs for private events have to be approved by you or a featured creator linked to the event. See more about this in the RSVP section.


Adding Artists

Use the artist search feature to add featured creators to an event. This will search for existing artist accounts on JamPad matching your input. If you see the artist you're trying to add, click on the search result to link them to this event.

Not all artists will have registered accounts (sadly). If you don't find the artist you're looking for, you can submit any text to show that they're featured without linking a JamPad account. You can tap on added artists to remove them and drag and drop to re-order them.


Adding Tags

Use the tag search feature to add any relevant tags to your post. Currently only recognized tags can be added, which will show up as you search for them. Double tap on added tags to remove them and drag and drop to re-order them.


Event Verification

If you are an admin account, all events you create will be automatically verified. If you are an artist account, any events you are a part of will be automatically verified. If you are a venue account, any events at your venue will be automatically verified.

Otherwise, your event will have to be verified by an admin or an artist featured at the event. When an event is verified, the user who posted it will be awarded 30 FrogCoin.
