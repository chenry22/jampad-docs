Event Feed

See the top upcoming events!

The event feed is a more algorithmic approach to event discovery. Here you can see the top upcoming events, sorted by a combination of factors including distance, date, and user interaction (likes and RSVPs).

At the top of the page is a toggle between "All" and "Following". The "All" tab will show you all public events, the "Following" tab will show you all events posted by or featuring accounts that you follow.

To the left of this toggle is a "Menu" tab, which opens a page listing a couple of different views (see the "Leaderboards" section for more about these).

To the right is a view toggler which allows you to switch between a one or two column view of the event feed. These views are the same except that the single column view shows you the start/end time of an event as well as the date.

Below this top tab are a series of time filters. Selecting these will show you only events in that time range (today, tomorrow, this week, etc).
